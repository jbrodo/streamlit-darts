# Darts with Streamlit demo

## Setup virtual env in root folder
```
python -m venv venv
```

## Activate venv
```
source venv/bin/activate
```

## Install poetry
```
pip install poetry 
```

## Install project
```
cd streamlit-darts
poetry install
```

## How to run
```
poetry run streamlit run streamlit_darts/runDarts.py
```
